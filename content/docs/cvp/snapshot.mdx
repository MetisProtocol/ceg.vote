Voting happens on Snapshot https://snapshot.org/#/metislayer2.eth

Community can vote YES, NO or ABSTAIN.

In order to pass the vote, a majority pass (80% YES) and quorum met (min. 10k METIS and 500 wallet addresses) is required as passing the community verification.

| Parameter                    | Value                             |
| ---------------------------- | --------------------------------- |
| Percent Approval             | 80%                               |
| Quorum                       | 5k METIS and 400 wallet addresses |
| Minimum Requirements to Vote | 0.1 Metis                         |
| Voting Strategy              | Quadratic Voting                  |
| Vote Timeline                | 72 hours                          |

<Callout>
  The value above may change based on community feedback, any change to the
  values will be communicated to the community
</Callout>

If a vote fails, the project can submit a new proposal.
